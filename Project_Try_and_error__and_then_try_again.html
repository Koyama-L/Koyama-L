TRIAL AND ERROR
We had to start somewhere, and then build
what we knew as we went along.
OBJECTIVE_
This project is about exploring typographic forms in new ways by doing different kinds of processes to types, leading to unexpected results. And by using the results, deliver three movie posters and a movie trailer.

APPROACH_
I use several different techniques to deal with the type and use the feeling from my dream as keywords. Some are colorful, some are quiet, some give the feeling of happiness, and some are scary. And I picked Gone Girl as my movie to deliver the three posters and trailer.
